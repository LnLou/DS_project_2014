Detailed Description

template<class T>
class ArrayList< T >

The ArrayList is just like vector in C++. You should know that "capacity" here doesn't mean how many elements are now in this list, where it means the length of the array of your internal implemention

The iterator iterates in the order of the elements being loaded into this list
