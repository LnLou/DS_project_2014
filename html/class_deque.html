Detailed Description

template<class T>
class Deque< T >

An deque is a linear collection that supports element insertion and removal at both ends. The name deque is short for "double ended queue" and is usually pronounced "deck". Remember: all functions but "contains" and "clear" should be finished in O(1) time.

You need to implement both iterators in proper sequential order and ones in reverse sequential order.
